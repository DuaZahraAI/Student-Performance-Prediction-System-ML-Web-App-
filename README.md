# 📊 Student Performance Prediction System (ML Web App)

## 🚀 Overview

This project is a Machine Learning Web Application built using Streamlit that predicts student academic performance based on input features such as study hours, attendance, and other academic indicators.

It demonstrates how a trained ML model can be deployed into a simple, interactive web interface for real-time predictions.

---

## 🎯 Problem Statement

Educational institutions often struggle to identify students at risk of poor performance early.

This project aims to:
- Predict student performance (Pass/Fail or Grade)
- Help in early intervention strategies
- Provide data-driven academic insights

---

## 🧠 Solution

A machine learning model is trained on student data and deployed as a web app using Streamlit, allowing users to input values and instantly receive predictions.

---

## ⚙️ Features

- 📥 Interactive user input form  
- 🤖 Machine Learning-based prediction system  
- ⚡ Real-time results  
- 📊 Prediction history tracking  
- 🎯 Simple and user-friendly UI  

---

## 🛠️ Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit
